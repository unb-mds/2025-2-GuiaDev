# **🏁** Sprint 10 - Padrões de qualidade & Fluxo de usuário

Período: 03/11/2025 a 09/11/2025

## Descrição

> Nesta sprint, o time focou na qualidade do código e na experiência do usuário. Trabalhamos na atualização da documentação (README, Documento de Visão, Contributing), otimização de banco de dados (cache e persistência de login social) e refinamento de interface (modal de configurações, login/cadastro e sidebar). Além disso, iniciou-se a implementação de testes de frontend.

## Objetivos

- Atualizar documentação do projeto (README, Documento de Visão, Code of Conduct).
- Implementar componente Modal de Configurações.
- Refinar design e funcionalidade das páginas de Login, Cadastro e Sidebar.
- Otimizar cache do banco de dados para atualizações dinâmicas.
- Corrigir bugs de armazenamento de usuários via Login Social.
